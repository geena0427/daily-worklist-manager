# 活跃任务记忆

这里用于记录当前仍需参与日常优先级判断的任务。

一般包括：
- 待处理任务（todo）
- 进行中任务（doing）
- 仍值得跟踪的阻塞任务（blocked）

这些任务会参与：
- 今日必做排序
- 挑战完成排序
- “再给我一件事”推荐
