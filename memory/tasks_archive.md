# 归档任务记忆

这里用于记录已归档的历史任务。

一般包括：
- 已完成任务（done）
- 已放弃任务（dropped）
- 经用户确认暂不参与日常排序的长期冻结任务

这些任务默认不参与：
- 今日优先级排序
- “再给我一件事”推荐

只有在以下情况下才需要查阅：
- 用户询问历史任务
- 需要确认是否做过类似事情
- 需要恢复旧任务
- 需要排查重复任务
